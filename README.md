# AccessLearn – AI-Powered Inclusive Learning Platform

AccessLearn is a Flask-based education technology project for IEEE TechForGood 2026. It supports two roles: Students and Teachers.

## Features
- Login and registration
- Student dashboard
- Teacher dashboard
- Teacher notes upload
- Student notes viewing
- AI-style chatbot with mock educational responses
- Notes summarizer
- Quiz generator with SQLite score tracking
- 3D glassmorphism responsive UI

## Run
```bash
pip install -r requirements.txt
python app.py
```
Open: http://127.0.0.1:5000

## Demo Tip
Register one Teacher account and upload notes. Register one Student account to view notes, summarize, chat, and attempt quizzes.
